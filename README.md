# SorveteBigData

ATIVIDADE DE CURSO DE ANÁLISE E DESENVOLVIMENTO DE SISTEMAS (SENAC-PE); MATÉRIA - BIG DATA

Imagine uma sorveteria que define o preço de seus sorvetes da seguinte forma:
 
Há dois tipos de sorvete: Comum, cujo preço é R$15 e Premium cujo preço é R$20 
O sorvete pode ser vendido em copinho ou casquinha. 
O copinho adiciona R$1 ao preço enquanto a casquinha adiciona R$2. 
A cobertura pode ser simples (apenas uma) custando R$1 ou especial (duas coberturas ou mais) custando R$2. 

• Crie o fluxograma/grafo para o cálculo do preço do sorvete e indique a complexidade ciclomática. 
• Implemente em alguma linguagem de programação
